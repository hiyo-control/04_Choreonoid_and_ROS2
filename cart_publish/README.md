# Cart publish
## How to build program
1. Open terminal at "./ros2_ws" folder
2. Execute "colcon build --packages-select cart_publish"

## How to execute
1. Open terminal at "./ros2_ws/intall"
2. Execute "source ./setup.bash"
3. Execute "ros2 run cart_publish Forward_command" or "ros2 run cart_publish Stop_command"
